# Telas implementadas

Esta pasta deve conter as quatro telas referentes às entregas dos protótipos de tela.

Abaixo estão as telas escolhidas e uma breve descrição de cada uma delas.

- [Tela inicial / Home](../frontend/index.html)
  - Tela de entrada do projeto Adote Lar, com seção de apresentação, explicação de funcionamento, parceiros e vitrine de animais. Inclui navegação para login, cadastro e detalhes dos animais.
- [Tela de detalhes do animal](../frontend/details.html)
  - Página de detalhe de um animal específico, com galeria de imagens, informações de porte, idade, localização, atributos e status de adoção/acolhimento.
- [Tela de login](../frontend/html/LoginPage.html)
  - Tela responsável por apresentar o formulário de autenticação, permitindo que o usuário informe e-mail e senha para entrar na aplicação.
- [Tela de cadastro](../frontend/html/RegisterPage.html)
  - Tela de registro para novos usuários, com seleção de perfil (protetor ou ONG), campos de dados pessoais e instruções de cadastro para começar a atuar na plataforma.
